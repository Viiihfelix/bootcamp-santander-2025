Estrutura básica

Todo documento HTML deve começar com esta declaração !DOCTYPE que diz ao navegador que tipo de documento é esse. Logo em seguida é preciso declarar a linguagem usada e para isso acrescenta-se lang="pt-br" que significa portugues, Brasil. O chamado metadado vem em seguida, seria o head, responsável pelos dados do HTML, nele podem contér elementos como:

  • title;
  • link;
  • style;
O head não aparece para o usuário, ao contrário do body, que vem logo em seguida. O body já é o corpo da página, é a parte que o usuário verá. Ou seja, é nele que irá conter parágrafos, imagens, videos, tabelas, links etc. São as tags que ajudarão com isso. Com o tempo, a semântica do HTML mudou, e usa-se dentro do body, o header (cabeçalho), o main (parte principal, o corpo) e o footer (rodapé).

O que são tags e seus atributos

As tags são elementos básicos que nos ajudam a estruturar nosso HTML, servindo para marcar e definir os tipos de conteúdo como textos, imagens, link e muito mais. Toda tag começa com sinal de menor que (<), sinal de abertura e termina com sinal de maior que (>), sinal de fechamento, porém acrescenta-se uma barra (/) que indica que a tag está sendo fechada. Alguns exemplos de tags:

  • nav: barra de navegação;
  • ul: listas não ordenadas (sem numeração);
  • ol: listas ordenadas (com numeração);
  • li: itens da lista (geralmente usadas dentro das tags ul e ol);
  • font: servia no HTML 4 para mudar a cor, tamanho e tipo da fonte diretamente no HTML, porém não deve mais ser usada, o ideal é formatar no CSS;
    • Exemplo: "font color="blue", size="2", face="Verdana"
    Resultado: Olá, mundo!
  • img: adiciona uma imagem. Obs: essa tag não precisa ser fechada;
  • a: âncora, adicionar links;
  • abbr: serve para indicar que uma palavra ou sigla é uma abreviação, e ao passar o mouse por cima, aparece um balãozinho com o significado
    • Exemplo: "abbr title="HyperText Markup Language">HTML"
    Resultado: HTML
Os atributos usados nas tags, dão informações extras, um exemplo é o uso do href onde define o link de destino quando se quer adicionar um link na sua página da web.

Formatando textos

No HTML, é possivel formatar seu texto utilizando tags de formatação, embora seja possivel também realizar isso no CSS. Alguns exemplos dessas tags são:

  • b: deixa o texto em negrito somente visualmente;
  • strong: deixa o texto também em negrito, porém de forma semântica, o que é melhor para acessibilidade;
  • i: deixa o texto itálico, visualmente;
  • u: deixa o texto sublinhado;
  • mark: deixa o texto destacado;
  • small: deixa o texto pequeno;
  • del: deixa o texto riscado ou removido;
  • sub: deixa o texto em subscrito(abaixo);
    • Exemplo: H2O
  • sup: deixa o texto em sobrescrito(acima);
    • Exemplo: 210

Como adicionar mídias

De forma bem simples, as tags utilizadas para adicionar mídias são intuitivas e utilizam atributos.

Imagens

As imagens usam a tag img que não tem fechamento. Dentro dela utiliza-se o src no qual indica o caminho ou o link da imagem. Porém, se por ventura, a imagem não carregar, pode-se utilizar o alt que seria um texto alternativo, tanto pra casos que a imagem não carregue ou até para leitores de tela.

Vídeos

Os videos usam a tag video e nesse é possivel utilizar o width para mensurar a largura do video, e o controls para mostrar os controles.

Áudios

Os áudios usam a tag audio e também, assim como nos videos, é possivel adicionar o controls para mostrar os controles (play, pause, volume). Para adicionar o caminho do arquivo, utiliza-se o source.

Mídias Externas

Caso queira adicionar mídias de locais externos, como Youtube, também é possivel, só utilizar o iframe.
